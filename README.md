# Drink Water Reminder (Desktop Notification)

A simple Python desktop application that reminds you to drink water every hour using system notifications and voice alerts.

This project is beginner-friendly and demonstrates how to create timed notifications in Python.

## Features
- Desktop notification every 1 hour
- Voice alert using Windows text-to-speech
- Runs continuously in the background
- Simple and easy to understand Python code


## Technologies Used
- Python
- plyer (for desktop notifications)
- win32com (for voice alerts)
- time module
